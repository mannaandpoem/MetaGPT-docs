# 解释器（MetaGPT Interpreter, MI）

## 什么是解释器

解释器是一个通过编写代码来解决问题的智能体。它能理解用户需求，制定计划，编写执行代码，并在必要时使用工具。这些能力使它能够处理广泛的场景，请查看下面的示例列表。
